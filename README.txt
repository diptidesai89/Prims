To run on eclipse IDE
1. Compile the following files in order : 
PQindex-> Graph -> PriorityQueueIndexed -> MST
2. Open the program MST.java in eclipse.
3. The program needs input file location as argument to the program in the run configuration option.
4. Run the program with the above configuration.
5. The output result is shown.




Specification of input file
<Number of Vertices> <Number of edges>
from:Vertex - to:Vertex - weight of the Edge
